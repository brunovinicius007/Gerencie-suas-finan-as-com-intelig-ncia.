# Minhas Finanças 🚀
> Um sistema de gestão financeira pessoal completo, inteligente e mobile-first.

**Minhas Finanças** é uma ferramenta robusta para quem deseja controle total sobre sua vida financeira. Muito além de uma simples planilha, o sistema utiliza inteligência local para dar conselhos, prever gastos e ajudar você a conquistar seus sonhos.

---

## 🌟 Diferenciais do Sistema

### 🧠 Inteligência e Análise
*   **Coach de Investimentos:** IA que analisa seu saldo e sugere como dividir seu dinheiro (Regra 50/30/20).
*   **Termômetro de Saúde Financeira:** Alertas dinâmicos baseados no comprometimento da sua renda.
*   **Analista de Cartão de Crédito:** Ranking detalhado de gastos (iFood, Streaming, Mercado) e alerta de limite de segurança (60%).
*   **Indicador "Dias de Vida":** Saiba exatamente quantos dias você consegue sobreviver com suas economias atuais.

### ⚡ Praticidade e Conectividade
*   **Importação de OFX:** Chega de digitar! Importe o extrato do seu banco (Nubank, Itaú, etc) e o sistema cadastra tudo sozinho.
*   **Relatórios em PDF:** Gere e baixe relatórios mensais detalhados com gráficos e tabelas.
*   **Resumo via WhatsApp:** Envie um relatório formatado para o seu WhatsApp com um clique.
*   **Lançamentos Recorrentes:** Cadastre despesas ou receitas para o ano todo de uma só vez.

### 📱 Experiência do Usuário (UX)
*   **PWA Ready:** Instale no seu celular como se fosse um aplicativo nativo.
*   **Modo Escuro Inteligente:** Ativação automática de tema Dark exclusiva para dispositivos móveis.
*   **Navegação Rápida:** Troque de mês e analise seu histórico com setas de navegação intuitivas.

### 🛡️ Segurança e Gestão
*   **Controle de Acesso:** Níveis de Administrador e Usuário Comum.
*   **Código de Convite:** Cadastro restrito para convidados (Código padrão: `nero`).
*   **Backup Total:** Exporte todos os seus dados em formato JSON para sua segurança.

---

## 🛠️ Tecnologias Utilizadas
*   **Backend:** PHP 7.4+
*   **Banco de Dados:** MySQL (com Transações e Integridade Referencial)
*   **Frontend:** Tailwind CSS (Design Moderno)
*   **Gráficos:** Chart.js
*   **PDF:** html2pdf.js
*   **Notícias:** Integração via RSS (G1 Economia)

---

## 🚀 Como Instalar (Localmente)

1.  Coloque a pasta do projeto no seu servidor local (Ex: Laragon ou XAMPP).
2.  Certifique-se de que o MySQL está ativo.
3.  Acesse pelo navegador: `http://localhost/seu-projeto/setup.php`
4.  O sistema criará o banco de dados e as tabelas automaticamente.
